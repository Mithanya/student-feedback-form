# Student Feedback Form

## Overview
This project is a basic Student Feedback Form developed using HTML. It is part of my initial learning phase in web development.

## Objective
The goal of this project is to understand how HTML forms work and how user input is collected and structured in a web page.

## Features
- Text input fields for user details
- Email input field
- Radio buttons for selection
- Dropdown for department selection
- Textarea for feedback input
- Simple and clean layout

## Technologies Used
- HTML

## Learning Outcomes
- Gained understanding of HTML structure
- Learned how to use form elements effectively
- Practiced building a simple real-world use case
